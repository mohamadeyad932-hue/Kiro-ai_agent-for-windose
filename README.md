# 🤖 الوكيل الذكي كيرو (Kiro Smart Agent)

نظام برمجي يعتمد على الذكاء الاصطناعي ومعالجة الصور، يهدف إلى أتمتة العمليات الصناعية أو البرمجية عبر تحليل الصور واستخراج البيانات بكفاءة عالية.

## 📌 المرحلة الأولى: وحدة معالجة الصور الأساسية

تم إنجاز المرحلة الأولى من المشروع بنجاح، والتي تتضمن بناء وحدة المعالجة المركزية للصور بالمميزات التالية:

- **استخراج البيانات الوصفية (Metadata):** قراءة أبعاد الصورة، صيغتها، وعدد القنوات اللونية بدقة للتعامل مع مختلف الصيغ مثل JPG و PNG.
- **التوحيد والمعالجة (Pre-processing):** تحويل الصور إلى التدرج الرمادي (Grayscale) لتقليل حجم البيانات وتسريع العمليات الحسابية.
- **البصمة الرقمية (Image Fingerprinting):** توليد بصمة فريدة لكل صورة باستخدام خوارزمية الهش الإدراكي `pHash`.
- **كشف التكرار (Duplicate Detection):** نظام دقيق لمقارنة مسافة هامنج (Hamming Distance) بين بصمات الصور لاكتشاف التطابق بنسبة تصل إلى 100%.

## 🛠 التقنيات المستخدمة

- **اللغة الأساسية:** Python 3.8+
- **معالجة المصفوفات والصور:** OpenCV (`cv2`) & Numpy
- **التعامل مع الصيغ:** Pillow (`PIL`)
- **خوارزميات البصمة:** ImageHash

🚀 كيفية التثبيت والتشغيل (Setup)
تحميل المشروع:

Bash
git clone https://github.com/your-username/Kiro-Smart-Agent.git
cd Kiro-Smart-Agent
تثبيت المكتبات اللازمة:

Bash
pip install -r requirements.txt
تشغيل الكود:
قم بوضع صورك في مجلد data/raw/ ثم شغّل الملف الأساسي:

Bash
python src/image_processor.py
📝 ملاحظات (Notes)
تم ضبط عتبة التشابه (Threshold) الافتراضية عند 95% لاعتبار الصور مكررة، ويمكن تعديلها من داخل الكود حسب الحاجة.
